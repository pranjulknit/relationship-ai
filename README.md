# Relationship AI MVP
Voice-based app for relationship insights using MERN, Vite, and GCP.

## Setup
### Backend
1. `cd backend`
2. `npm install`
3. Set `.env`
4. `npm run server`

### Frontend
1. `cd frontend`
2. `npm install`
3. Set `.env`
4. `npm run dev`

## Features
- 4-phase conversation flow
- Voice input/output with GCP
- MongoDB Atlas storage
- Modular backend
